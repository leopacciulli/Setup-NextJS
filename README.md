<h3 align="center">
  Setup React.js using Next.js, TypeScript, ESLint and Styled Components
</h3>

<br><br>

````zsh
# to run the project
$ git clone https://github.com/leopacciulli/Setup-NextJS.git

# cd folder
$ cd reactsetup

# install dependencies
$ yarn

# start on port 3000
$ yarn dev
````
